## 항공물류 AirwayBill 통합 서비스
- 공항 주변 화물 운송 현장에서 AWB(Air Waybill) 문서에 대한 의존도가 높지만 해당 문서의 생성 및 관리 과정에서 세관, 항공사, 화주 간 데이터 연계가 부족
- 따라서 이의 데이터를 통합 관리하고, AWB 문서 작성 및 관리 프로세스를 디지털화하여 효율적이고 신뢰할 수 있는 시스템을 개발하고자 함.
